SensoPatch

  • Skills

    - Flex PCB design on KiCAD
    - ARM cortex-M4F microcontroller
    - Bluetooth LE (6 peripherals) - SMD soldering

SensoPatch is a wireless, reconfigurable haptic feedback system designed for study on upper limb amputees with EMG-controlled prosthetic hand. The system consists of a tactile pressure sensing glove and wireless vibration patches.

Design

The Glove

  • 5x5 sensor array on fPCB: 5 conductive column traces and 5 conductive row traces intertwine at major sensory locations in the fingers and palm
  • Zig-zag trace design allowing fPCB to stretch to facilitate flexible movement of the thumb
  • Features XIAO BLE nRF52840, lithium-ion battery and on/off switch
  • Once the Velostat on the fPCB is pressed, the pressure at each intersection directly translates to voltage readings. The pressure on the hand is measured via TDMA approach.

    • The Patches

    • Each features XIAO BLE nRF52840, vibration motor, boost circuit, lithium-ion battery, and on/off switch on a 1.36” diameter PCB.
    • Can be placed anywhere on the body. The modularity of each feedback motor allows users to easily alter the number of motors and customize motor placement.

    How It Works

    The tactile sensor array collects and wirelessly transmits pressure information to a set of haptic feedback patches that convert the sensor data into vibrotactile sensations for users.

    Since there are more number of sensors than the number of feedback motors, we can program different methods of compressing tactile data into feedback by mapping sensors on different areas of the hand to the vibration motors, as shown in the table. We can also vary the number of the motors and stimulation location to find the optimal configuration that delivers efficient feedback without confusing the users.

    Project Goals

    SensoPatch is a reconfigurable, open-source tool that allows future research to conduct studies on various aspects of the optimal method of haptic feedback delivery. As part of the preliminary study, participants were recruited to complete blindfold discrimination and functional tasks using controlling myoelectric hand via EMG and bypass prosthetic.
